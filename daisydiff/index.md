#### Welcome. To use this DaisyDiff clone, add it to Maven dependencies of your project.

Like this:

~~~~
<dependency>
    <groupId>io.github.valters</groupId>
    <artifactId>daisydiff</artifactId>
    <version>{{site.data.daisydiff.version}}</version>
</dependency>
~~~~
